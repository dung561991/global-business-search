# Global Business Search

## Overview
Global Business Search is an innovative platform designed to facilitate comprehensive searches of business information across multiple databases. It aims to simplify the process of finding relevant business data for users in various industries.

## Features
- **Multi-database Search**: Access information from various databases with a single query.
- **User-friendly Interface**: Intuitive interface that simplifies the search process.
- **Real-time Updates**: Stay informed with the latest business information and updates.
- **Data Visualization**: Visualize search results for better insights.
- **Customizable Filters**: Apply filters to narrow down search results based on specific criteria.

## Tech Stack
- **Frontend**: React.js
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Search Engine**: Elasticsearch
- **Hosting**: AWS

## Installation Guide
1. Clone the repository:
   ```bash
   git clone https://github.com/dung561991/global-business-search.git
   ```
2. Navigate to the project directory:
   ```bash
   cd global-business-search
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Set up the environment variables (create a `.env` file in the project root and add your settings).
5. Start the application:
   ```bash
   npm start
   ```

## Usage Instructions
Once the application is running, navigate to `http://localhost:3000` in your web browser. From there, you can use the search bar to begin finding business information. Utilize the provided filters to refine your search results and explore the data visualizations for better insights.
